# FACTORY

Agent Factory — verified (harness x model) pairing production. Layer 3 of
PAO -> HSS/hux -> Factory. Pre-verifies pairings with mechanical probes over an
isolated PAO bus, records match-ledger entries with copied evidence, and gates
binding on verified pairings only.

Design source: PAO repo `_workspace/DESIGN-AgentFactory.md` (Korean, 15-finding
red-team applied); condensed binding design in `.pgf/DESIGN-FACTORY.md`.
